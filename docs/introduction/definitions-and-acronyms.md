# Definitions and Acronyms

| Term | Definition |
|------|------------|
| SRS | Software Requirements Specification. A document that describes what a software system should do, its functionality, and any constraints. |
| MFA | Multi-Factor Authentication. A security process that requires users to provide two or more verification factors to gain access to an account. |
| Browser Fingerprinting | A method of collecting information about a remote computing device for the purpose of identification. Information includes browser type, version, operating system, plugins, and other settings. |
| Device ID | A unique identifier assigned to a user's device or browser session, used in combination with browser fingerprinting to recognize a specific device. |
| IP Geolocation | The process of determining the geographic location of a device by its IP address. |
| Trusted Device | A device or browser that has been explicitly confirmed by the user as legitimate, preventing future new-login notifications from it. |
| View Mode | A temporary, restricted account state where a user can only view information but cannot perform any transactional actions (e.g., create, edit, delete). |
| Deactivated Account | An account that has been disabled due to prolonged inactivity, preventing the user from logging in. The user's data is preserved but access is revoked. |
